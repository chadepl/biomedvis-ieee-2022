FoldVis

This website shows the progress in the FoldVis project for the BioVis challenge. There are 3 types of content:

  • Diary: weekly entries showing some progress and thoughts that arise when exploring the data and trying out different visualizations.
  • Paper: links to the latest iteration of the 2-page paper that details the design process and prototype.
  • Prototype: links to the latest iteration of the prototype. Some ideation iterations for the prototype can be found at this link.
